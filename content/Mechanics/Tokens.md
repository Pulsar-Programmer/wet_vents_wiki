Interaction Tokens or Tokens for short are a system where, when you interact, you exchange a token. You write on a piece of paper or anything and give them a code to interact and cause an Action.

The tokens are 6-digits long and alphanumeric. They are given to you by the [[Codes]] command when everyone has started. Each code does something special which is also stated per role.

Tokens are created to preserve LOCALITY. Exchanging Tokens means you have to be close to the target. These are essential for all imposters, some neutrals, and some crewmates.

There are only two regulating rules:
- Do not exchange codes you've received/given with other people.
- Digitally exchanging codes does not count.

At the end of the day, codes must be reported via the following commands along with some information:
[[Give]], if you are giving a code.
[[Receive]], if you are receiving a code.
