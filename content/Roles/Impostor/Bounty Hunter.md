The Bounty Hunter gets a specific target and can kill only that target. Once the target is killed, two new ones are given the following day.

Specifically, they are given a code that can only be used to kill the target. The number of targets they kill successfully they earn that number more kill tokens.

**Config**
`new_targets: u8 = 2`: How many new targets are given after a target is killed.
`non_bounty_max_kill: u8 = 0`: How many players can be killed unrelated to the given target per day.
